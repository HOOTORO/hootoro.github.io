### Rating

|        Grade | Meaning                          |
| -----------: | -------------------------------- |
|      **GOD** | Like!                            |
|     **Good** | Choice to make.                  |
| **Sometime** | Take a place, average.           |
|     **Sour** | Taste, rarely desired.           |
|      **Icy** | Fresh, mentol, etc.              |
|      **Bad** | Experience                       |
| **Chemical** | Class today! Don't miss          |
|   **Exotic** | Extremely rare and unusual taste |
|     **Acid** | Too sour                         |
